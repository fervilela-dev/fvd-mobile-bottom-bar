# fvd-mobile-bottom-bar

Versión 1.2.0

Plugin de WordPress para agregar navegación por botones estilo app, para responsive web.

## Novedades FerVilela
- Panel de administración bajo el menú **FerVilela → Barra inferior**.
- Configura cantidad de botones (hasta 6), URL, color, tamaño e ícono (Dashicons).
- Define tipo de acción por botón: enlace, abrir buscador, abrir mini carrito o trigger personalizado.
- Integra botones con Elementor / Royal Elementor Addons usando selector sugerido o selector CSS manual.
- Control de vistas: solo móviles, solo tablet, solo desktop o rango personalizado por ancho (px).
- Color de fondo personalizable para la barra.

## Changelog
- 1.2.0: Botones con tipo de acción configurable (URL, buscador, mini carrito, trigger custom), integración directa con widgets de Elementor y Royal Elementor Addons, fallback por URL, soporte de selectores personalizados, disparo vía JS sin recargar página y campo de etiqueta accesible opcional.
